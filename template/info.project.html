Это страница, на которой мы выводим общедоступную информацию по проекту.

Здесь живут ссылки на имеющиеся карты, описание проекта.

Для владельца проекта показываются кнопки настроек:
а) проекта
б) прав пользователей

Для остальных зареганных (не редакторов) показывается кнопка join team

кроме того, для редакторов и владельца показывается changelist по проекту

    А пока что тут живет просто ссылка на активную карту
  • Единственная карта
  • Настройки прав команды (ссылку видит владелец)
  • Настройки проекта (ссылку видит владелец)
  • Настройки участия (ссылку видит редактор)
